Bot Helade
===============

What?
---------
Simple echo bot to be run on python.

How to run?
------------
$: python bicho.py

Logic?
------------
Tell the bot to salute you:
$: python bicho.py

Then tell him something by running the ia:
$: python yo.py 
... then, do the stIn.

----------
